A little bit about this blog.

This blog doesn't need the attention of readers. It was created for one purpose - learning English. Of course, if you want to know a little bit about the author, you can do that here. But the first idea was to write scripts for movies, books and games. Then I decided to add a topic where I can talk about my favorite works of art, as well as share my non-expert opinion. WARNING: Since this project was created for learning English, it is forbidden to use a translator. 😄
